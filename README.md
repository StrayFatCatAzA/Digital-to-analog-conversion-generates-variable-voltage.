Digital-to-analog conversion generates variable voltage
数模转换产生可变电压
题目13：数模转换产生可变电压
功能描述：使用STM32的DAC（或利用定时器产生高精度PWM经滤波后模拟DAC）输出一个可调的直流电压（0-3.3V）。通过两个独立按键控制输出电压值：一个按键步进增加电压值（步长0.1V），一个按键步进减小电压值。当前设置的电压值（如“2.5”）实时显示在2位数码管上。
实现指标：
DAC/PWM输出电压稳定，纹波小。
输出电压范围0-3.3V，步进控制精确。
数码管显示的设置电压与实际输出电压基本吻合。

使用hal库编写，电压生成采用pwm生成，主控使用f103c8t6，数码管采用4位8段数码管，按键采用4x4矩阵键盘，显示屏采用0.96oled（不是必须的，调试用）
接线：
矩阵键盘 -- 行1 -- PA0
矩阵键盘 -- 行2 -- PA1
矩阵键盘 -- 行3 -- PA2
矩阵键盘 -- 行4 -- PA3

矩阵键盘 -- 列1 -- PA4
矩阵键盘 -- 列2 -- PA5
矩阵键盘 -- 列3 -- PA6
矩阵键盘 -- 列4 -- PA7
